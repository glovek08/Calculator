BASIC CALCULATOR

The design of this basic calculator is roughly based on the Sanyo CX-8001 model introduced in 1973, it's a clear depiction of early 70's design, looking both retro and futuristic with that beautiful VFD tube.

The Sanyo CX-8001 is a basic eight-digit portable calculator which uses exactly the same NEC 271D processor and support chips as the desktop ICC-805.

The keyboard has the same 18 keys, but in a more compact arrangement. The display uses an eight-digit double-ended vacuum fluorescent tube instead of the eight individual tubes of the earlier models.

An unusual feature is the use of a pair of (then very new) light-emitting diodes for the negative and overflow indications.

The battery-operated power supply draws about 120 to 150mA (about 700-900mW) from four disposable AA cells or an external 6V adaptor (Sanyo type CU-8001E).
